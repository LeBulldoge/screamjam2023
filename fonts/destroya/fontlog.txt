FONTLOG for the Destroya font

This file provides detailed information on the Destroya Font Software.
This information should be distributed along with the Destroya font and any derivative works.

Basic Font Information

Destroya is a Unicode typeface that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
